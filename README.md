# Solana-Claim-Web-Drainer
Сайт позволяющий получать сид фразы пользователей

Работает с помощью фейк модальный окон и транзакции, с помощью которых сид фраза пользователей отправляется в вашего telegram бота

Полный код со всеми нужными функцииями бесплатно, вам нужно только вставить в код токен вашего телеграм бота и ваш айди. 

/////

A website that allows you to obtain user seed phrases.

It works using fake modal windows and transactions, through which the user's seed phrase is sent to your Telegram bot.

The complete code with all the necessary functions is free; you only need to insert your Telegram bot's token and your ID into the code.
